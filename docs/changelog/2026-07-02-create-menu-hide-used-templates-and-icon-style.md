## Что сделано

### Скрытие уже использованных шаблонов (`TaskListView`)

В режиме проекта из меню создания треда прячутся шаблоны, по которым в проекте
**уже есть живой (не удалённый) тред** — чтобы не плодить дубликаты. Признак —
`source_template_id` на `project_threads`. Удалишь тред → шаблон снова появляется
в меню.

- `usedTemplateIds` — set `source_template_id` живых тредов проекта (только
  `isProjectMode`).
- `threadTemplates` фильтруется: `allThreadTemplates` минус использованные.
- В глобальном режиме (без проекта) фильтр не применяется.

### Стиль иконки шаблона в меню (`CreateThreadButtonGroup`)

- Иконка шаблона: раньше — белый глиф в цветной плашке-квадрате (`ACCENT_BG` +
  `bg-*`), теперь — просто цветной глиф (`ACCENT_TEXT` + `text-*`), компактнее.
- Попап списка шаблонов сделан прокручиваемым:
  `max-h-[min(70vh,28rem)] overflow-y-auto` — не упирается в край экрана при
  большом числе шаблонов.
- Уплотнены отступы строк (`py-1.5` → `py-1`).
