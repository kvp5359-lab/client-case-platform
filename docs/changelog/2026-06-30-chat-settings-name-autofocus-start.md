# Автофокус названия в настройках чата — каретка в начало

**Дата:** 2026-06-30
**Тип:** fix
**Статус:** completed (ждёт деплоя фронта)

---

> Правка из параллельной сессии, вкоммичена вместе с мобильной шапкой треда.

## Что было

В диалоге настроек/создания чата (`ChatSettingsDialog`) поле «Название»
авто-фокусировалось через атрибут `autoFocus`. У длинного названия каретка
вставала в конце строки, и input показывал хвост названия, а не начало.

## Что стало

`autoFocus` заменён на эффект (`useEffect` + `requestAnimationFrame`): при
открытии (в edit-режиме или для не-email) ставим фокус, `setSelectionRange(0, 0)`
и `scrollLeft = 0` — каретка и прокрутка в НАЧАЛЕ, видно начало названия.
