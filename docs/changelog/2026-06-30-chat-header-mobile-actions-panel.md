# Мобильная шапка треда: выдвижная панель действий + индикатор канала в неё

**Дата:** 2026-06-30
**Тип:** feat
**Статус:** completed (ждёт деплоя фронта)

---

Часть мобильной адаптации. Всё — фронт (CSS/классы/портал), через `md:`-брейкпоинт.
Десктоп не затронут.

## Проблема

В шапке треда (`TaskPanelTaskHeader`) на узком экране слишком много инлайновых
действий (статус, имя, иконка, исполнители, срок, проект, поиск, индикатор
канала, ⋮) — название сильно обрезалось.

## Что сделано

### Выдвижная панель действий (мобила)
- Ряд 1 (всегда): статус + **имя на максимум ширины** + поиск + ⋮ + кнопка-шеврон
  `ChevronDown` + крестик.
- По тапу на шеврон раскрывается панель снизу (абсолютная, поверх ленты) с
  «тяжёлыми» действиями: исполнители / срок / проект. Шеврон поворачивается на 180°.
- Реализация без дублирования: группа действий обёрнута в `md:contents` —
  на десктопе обёртка «растворяется», действия inline в строке как раньше; на
  мобиле это абсолютная панель (скрыта, пока не открыта). `toolbarRef` (портал
  `ChatToolbar`) — один экземпляр.

### Подписи «Срок»/«Проект» в панели
В панели есть место — показываем подписи: у `DeadlinePopover` добавлен проп
`placeholderLabelClassName` (в шапке `""` → «Срок» виден всегда; в списках задач
по-прежнему `hidden md:inline`); у `ChatSettingsProjectSelector` убран
`labelClassName="hidden md:inline"`.

### Индикатор канала → в панель (мобила)
Поиск остаётся наверху, индикатор канала уходит в выдвижную панель (справа). Чтобы
не дробить портал `ChatToolbar`, индикатор канала **дублируется** (он
stateless): inline на десктопе (`hidden md:inline-flex`, рядом с поиском) + портал
в слот шапки на мобиле (`md:hidden`-слот, виден только на мобиле). Проброс:
`ChatToolbar` получил `channelContainer`; `MessengerTabContent` —
`channelPortalContainer`; `TaskPanel` держит второй контейнер и связывает слот в
`TaskPanelTaskHeader` (`channelToolbarRef`) с порталом. Email-ветка не дублируется.

### Прижатие иконок вправо
Убран лишний второй `ml-auto` у шеврона (два `ml-auto` делили свободное место →
зазор). Теперь поиск/канал/⋮/шеврон/× кластером справа.

## Грабли

- `md:contents` — приём «обёртка без бокса на десктопе»: позволяет одной и той же
  группой быть inline (десктоп) и абсолютной панелью (мобила) без дублирования.
- Индикатор канала из портала `ChatToolbar` дублируется CSS-видимостью (inline
  `hidden md:inline-flex` + портал в `md:hidden`-слот) — для stateless-иконки ок,
  для stateful (email `EmailSubjectBar`) НЕ дублировать.
