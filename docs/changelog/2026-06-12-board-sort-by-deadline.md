# Сортировка колонки доски по сроку

**Дата:** 2026-06-12
**Тип:** feature
**Статус:** completed (ждёт смок-теста в проде)

---

## Что было

В меню колонки доски (`board_lists`) для списков задач не было способа
быстро упорядочить карточки по дедлайну — только ручной drag или общие
режимы сортировки списка.

## Что стало

В меню заголовка списка (только для `entity_type='thread'`) появился пункт
**«Сортировать по сроку»** (иконка `CalendarClock`). Разовое действие:

- Задачи **с дедлайном** уходят вперёд, по возрастанию срока (с учётом времени).
- Задачи **без дедлайна** сохраняют текущий относительный порядок и идут следом.
- Результат пишется как **ручной порядок** (`manual_order`) через
  `useReorderBoardListItems`. Если список был не в режиме `manual_order` —
  он переводится туда (`useUpdateList`), иначе сохранённый порядок
  проигнорировался бы.

Действие разовое: дальнейшие правки дедлайнов автоматически не пересортировывают —
нужно нажать пункт снова.

## Файлы

- `src/components/boards/BoardListHeader.tsx` — пункт меню + проп `onSortByDeadline`.
- `src/components/boards/hooks/useBoardListCardSetup.ts` — логика `sortByDeadline`.
- `src/components/boards/BoardListCard.tsx` — проброс в заголовок.

## Ограничения

- Только для списков задач (`thread`), не для проектных/inbox.
- Сортировка разовая (snapshot), не «живой» режим — по аналогии с ручным DnD.
