# Время открытия в «Недавнем» глобального поиска

## Что сделано
В выпадашке поиска (сайдбар) у строк раздела «Недавнее» справа показывается,
когда объект открывали: сегодня → ЧЧ:ММ, вчера → «вчера», раньше — короткая
дата. Формат тот же, что у времени во «Входящих» (переиспользован `formatTime`
из inboxChatItem.helpers).

Данные уже были: RPC `get_recently_viewed` всегда возвращала `opened_at`,
просто UI её не показывал. В режиме поиска время видно у строк секции
«Недавнее» (подтягивается из кэша недавнего); у остальных результатов
поиска времени нет — они не открывались.
