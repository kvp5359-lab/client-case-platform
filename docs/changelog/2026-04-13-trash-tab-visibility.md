# Корзина — видимость вкладки для администраторов

**Дата:** 2026-04-13
**Тип:** fix
**Статус:** completed

---

## Что сделано

### 🟠 Вкладка «Корзина» не отображалась владельцу воркспейса

- **Проблема:** вкладка «Корзина» в настройках воркспейса была видна только при `isOwner = true`. Но у владельца (kvp5359@gmail.com) в БД была только роль «Администратор» без роли «Владелец» в массиве `workspace_roles` — поэтому вкладка не отображалась.
- **Решение:** добавлена роль «Владелец» участнику в БД.

### 🟡 Администраторы не видели вкладку «Корзина»

- **Проблема:** доступ к корзине был ограничен только владельцем (`isOwner`). Администраторы с правом `manage_workspace_settings` не могли управлять удалёнными элементами.
- **Решение:** условие показа расширено — вкладка видна владельцу ИЛИ любому с правом `manage_workspace_settings`.
