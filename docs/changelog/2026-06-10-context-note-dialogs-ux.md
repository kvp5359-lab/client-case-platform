## Что изменилось

Диалоги внутренних заметок контекста проекта (создание и редактирование)
переработаны по UX:

1. **Кнопка «Сохранить» перенесена в шапку** диалога (рядом с названием), нижний
   футер убран — удобнее при длинном тексте, не нужно скроллить вниз.
2. **Подтверждение при закрытии с несохранёнными изменениями** — при клике вне
   диалога, по крестику или Esc, если есть правки, показывается AlertDialog
   «Закрыть без сохранения?» («Продолжить редактирование» / «Закрыть без
   сохранения»). Защита от случайной потери текста.
3. **Макс. высота редактора** (`max-h-[calc(100vh-…)]`) — длинная заметка не
   распирает диалог за пределы экрана, появляется внутренний скролл.
