# Профиль: интеграции и настройки — компактный аккордеон

**Дата:** 2026-06-09
**Тип:** feat (UI)
**Статус:** completed

Страница профиля была длинной лентой крупных раскрытых карточек (Google Drive,
Gmail, Telegram, WhatsApp, Перевод, Настройки приложения) — много пустого места,
долгий скролл. Переработана в два компактных раздела со сворачиваемыми строками.

---

## Что изменилось

**Единая строка-аккордеон.** Новый компонент
[`IntegrationRow`](../../src/page-components/ProfilePage/IntegrationRow.tsx) —
сворачиваемая строка: иконка + название + статус-точка (зелёная «подключено» /
жёлтая «требуется переподключение» / серая «не подключено») + шеврон. По умолчанию
свёрнуто, детали разворачиваются по клику. На базе Radix Collapsible.

**Раздел «Интеграции»** (одна карточка, строки через разделители):
- Google Drive — статус + кнопки в развороте.
- Email (Gmail) — статус по аккаунтам (warn при отозванном доступе → раскрыт
  сразу), в развороте: аккаунты + адрес пересылки + кнопки.
- Личный Telegram — статус «подключено» по активной MTProto-сессии или Business-
  подключению текущего юзера; в развороте — `PersonalTelegramSection selfOnly`.
- WhatsApp / Instagram (Wazzup) — строка только если есть назначенные номера;
  в развороте read-only список.

**Раздел «Настройки»** (вторая карточка, аналогично):
- Перевод сообщений — статус = текущий язык.
- Настройки приложения — статус = тема; в развороте модель AI / тема / уведомления
  / сохранение.

**Прочее:** унифицированы заголовки секций (`text-base`), отступы и ритм между
карточками.
