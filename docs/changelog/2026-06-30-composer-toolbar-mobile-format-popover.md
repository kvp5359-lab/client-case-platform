# Композер: адаптивный ряд инструментов — кнопки прячутся по нехватке ширины

**Дата:** 2026-06-30
**Тип:** fix
**Статус:** completed (ждёт деплоя фронта)

---

## Проблема

Ряд форматирования в композере (`MessageInputToolbar`) на узком ряду налезал на
кнопки отправки. Левая группа `flex-1 min-w-0`, кнопки внутри не сжимаются →
переполняют и накладываются на правую группу. `overflow-x:auto` использовать
НЕЛЬЗЯ — он делает `overflow-y:auto`, что обрезает инлайновые всплывашки вверх
(QuickReplyPicker) — была регрессия. Подход «прятать только на мобиле» (`md:`) не
решал узкую десктоп-панель (её можно ужать перетаскиванием).

## Что сделано

### Адаптивно по ширине РЯДА (container query, не вьюпорт)
Ряд инструментов (`.composer-tools-cq`) сделан container-контекстом. Кнопки
форматирования (`MessengerToolbar`: B/I/U/S/цитата/нумерованный/маркированный)
прячутся **по одной, справа налево**, по мере сужения ряда — `@container
composer-tools (max-width: …)` с порогами на каждую `nth-child`-кнопку. Работает
и на мобиле, и в узкой десктоп-панели.

### Непоместившиеся — в всплывающую панель «Aa»
Как только хоть одна кнопка не влезла (ряд ≤ 350px), появляется кнопка **«Aa»**
(`Type`, `.composer-fmt-toggle`), открывающая полную панель форматирования
всплывающей панелью вверх (Radix `Popover` side="top" — портал, не клиппит).
`MessengerToolbar` stateless (работает с `editor`) → две копии (inline + popover)
безопасны.

### Кнопка статуса задачи
`TaskStatusPicker` («Новый статус») — широкая, прячется первой (`@container … (max-width:
400px)`); статус доступен в шапке/меню «⋮».

## Грабли

- На тулбар с инлайновыми всплывашками (`absolute bottom-full`) НЕЛЬЗЯ вешать
  `overflow-*` — обрежет по вертикали. «Кнопки не влезают» решать container-query
  + портал-поповер, не клиппинг-скроллом.
- `nth-child`-скрытие завязано на структуру `MessengerToolbar` (7 Toggle-кнопок
  прямыми детьми div). При изменении набора — поправить пороги.
- Пороги (`max-width`) подобраны под ~28px/кнопку — ориентир, не пиксель-перфект;
  «Aa» — полный фолбэк в любом случае.

## Проверки

tsc 0, lint 0. Десктоп с широкой панелью — всё inline как раньше. Смок: сужать
правую панель — кнопки форматирования исчезают по одной, появляется «Aa» с полным
набором; на мобиле то же; отправка не перекрыта.
