# Подсветка группы задач тенью при наведении

При наведении курсора на группу задач (вкладка «Задачи» проекта) вокруг всей
группы появляется лёгкая тень (тонкое кольцо + мягкая тень, transition 150мс) —
видно, где группа начинается и заканчивается. Без ховера вид не менялся.
