# Документы: на десктопе пропали кнопки «Слот»/«Документы» и иконки комментариев

**Дата:** 2026-07-07
**Тип:** fix (фронт-CSS)
**Статус:** ждёт деплоя фронта

---

**Симптом (прод, десктоп):** на вкладке документов проекта пропали кнопки загрузки
документов в группу и добавления слота (шапка папки), а также иконка добавления
комментария напротив документов.

**Корень:** регрессия из коммита `b588e18` («fix(mobile): hover-действия видны на
тач»). Он массово навесил префикс `md:` на классы, чтобы действия были видны на
тач-экранах. Для **opacity** это верно (`opacity-0 group-hover` → `md:opacity-0
md:group-hover` = мобила всегда видит, десктоп по ховеру). Но заодно испортил
**display-классы**: `hidden md:flex` → `md:hidden md:flex`.

`md:hidden md:flex` — сломанная логика: на десктопе оба варианта применяются на `md+`,
и `md:hidden` (в Tailwind `.hidden` эмитится позже) **побеждает** `md:flex` →
`display:none` всегда, даже при ховере. Элемент исчезал на десктопе.

Затронуло 3 места:
- кнопки **«Слот» / «Документы»** в шапке папки — [`FolderCardHeader.tsx:132`](../../src/components/documents/Documents/FolderCardHeader.tsx)
- иконка комментария у папки — [`FolderCardHeader.tsx:87`](../../src/components/documents/Documents/FolderCardHeader.tsx)
- иконка комментария у документа — [`DocumentItem.tsx:268`](../../src/components/documents/Documents/DocumentItem.tsx)

**Фикс:** в 3 местах `md:hidden` → `hidden` (восстановлено «спрятать на мобиле» —
там своя кнопка «+»; opacity-ховер для тача оставлен). Больше нигде в `src` этого
паттерна нет (проверено grep'ом `md:hidden md:flex` / `md:hidden md:inline-flex`).

## Грабли (на будущее)

Префикс `md:` уместен для **opacity** hover-действий (touch-видимость), но НЕ для
**display**-классов вида `hidden md:flex`. `md:hidden md:flex` прячет элемент на
десктопе (`hidden` побеждает `flex`). Правильный «мобила скрыта / десктоп показан» —
базовый `hidden` + `md:flex`, не `md:hidden md:flex`.

## Проверки

tsc 0, lint 0. Чисто CSS-классы; поведение на мобиле не менялось (там «+»-меню).

**Файлы:** `DocumentItem.tsx`, `FolderCardHeader.tsx`.
