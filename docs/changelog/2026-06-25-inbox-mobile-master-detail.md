## Мобильный инбокс (master-detail)

`InboxPage/index.tsx` + `InboxSidebar.tsx`:

- На мобиле список чатов — на весь экран; при выборе чат открывается
  **выезжающим справа оверлеем** (translate, 300мс, тень слева), слева остаётся
  узкая полоска списка (`left-[28px]`). На десктопе — прежний двухпанельный режим.
- Контент чата держится ещё 300мс после закрытия (`renderChat` с задержкой),
  чтобы был виден слайд выезда.
- На мобиле не авто-выбираем первый чат (`isMobileViewport()` → `null`), вход в
  чат — тапом.
- `InboxSidebar`: пропы `className`/`narrow` — узкая полоска (шапка списка скрыта)
  при открытом чате на мобиле; на десктопе всё как было.

## Слайд-анимация боковой панели

`TaskPanelTabbedShellRenderer.tsx`: панель проигрывает слайд **выезда**
(`render` держится 300мс после закрытия), длительность перехода 200→300мс —
единый темп с чат-панелью инбокса.

## UX мессенджера

- `InboxChatItem.tsx`: имя отправителя в превью «Входящих» — первое слово имени
  (Telegram-style, без фамилии); для email-адреса — локальная часть до «@».
- `MessageBubble.tsx`: двойной клик по пустому месту ряда (сбоку от бабла) =
  «Ответить» на сообщение; гасится случайное выделение/попап «Цитировать».
