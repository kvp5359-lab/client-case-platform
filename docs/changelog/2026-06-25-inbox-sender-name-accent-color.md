# Цвет имени отправителя во «Входящих» = акцент треда

**Дата:** 2026-06-25
**Тип:** style
**Статус:** completed (ждёт деплоя фронта)

---

> Правка из параллельной сессии, вкоммичена вместе с мобильной адаптацией документов.

## Что было

Имя отправителя в превью списка «Входящие» (`InboxChatItem`) красилось
фиксированным синим `#337acc` (`SENDER_NAME_CLASS = 'font-normal text-[#337acc]'`),
независимо от цвета самого треда.

## Что стало

Имя отправителя берёт **цвет акцента треда** (`accent.text`) — совпадает с цветом
иконки/бейджа этого треда. Константа `SENDER_NAME_CLASS` теперь несёт только вес
(`font-normal`), а цвет добавляется на месте: `cn(SENDER_NAME_CLASS, accent.text)`.
Применено во всех трёх ветках превью (реакция, сообщение, медиа).
