# 2026-07-21 — Композер: пикер по «/», ресайз высоты + иконка канала в поиске

## Пикер быстрых ответов по клавише

- `/`, `\`, `|` открывают пикер быстрых ответов (как кнопка-молния), когда
  символ идёт в **начале поля/строки или сразу после пробела** (проверка
  `\s$` по тексту текстблока до каретки, hardBreak → `\n`). Если прямо перед
  символом непробельный символ («км/ч», «C:\») — вставляется как обычно.
- Ловим в ProseMirror-шорткате (`SendOnEnter`), а не на всплытии React-события:
  редактор обрабатывает keydown первым, поэтому надёжно гасим вставку символа и
  открываем пикер. Прежний обработчик на wrapper-div не срабатывал.

## Попап пикера — над всем композером

- Попап якорился к обёртке кнопки-молнии внутри тулбара (`.composer-tools-cq` —
  container-query контейнер, он же containing block для absolute), из-за чего
  `bottom-full` вставал над кнопкой и перекрывал поле ввода.
- Теперь попап **портится (React Portal) в корень композера** (`[data-composer-root]`
  на `MessageInput`, у него `relative` и нет `container-type`) → `bottom-full`
  встаёт над ВСЕМ композером, устойчиво к высоте поля. Click-outside учитывает
  портанутый попап; фолбэк на inline, если корень не найден. Высота списка/вкладок
  адаптивна (`h-[min(400px,45vh)]`).

## Ресайз высоты поля ввода

- Ручка резайза теперь задаёт **базовую (min) высоту** поля, а не только потолок
  роста — поле можно увеличить даже пустым. Контент авто-растёт до `max(база, 255)`,
  дальше скролл.
- Потолок — **высота окна** (`window.innerHeight − 150`, динамически), без фикс-600.
- Двойной клик по зацепу — **+500px** (в тех же рамках окна).

## Иконка канала треда в глобальном поиске и «Недавнем»

- `EntityIcon` для чат-тредов рисовал generic-облачко вместо иконки канала
  (Telegram-тред → облачко вместо самолётика). RPC `global_search` и
  `get_recently_viewed` не отдавали `project_threads.icon`.
- Добавлено поле `thread_icon` в обе RPC (миграция `20260721130000`,
  DROP+CREATE, гранты сохранены); `EntityIcon` резолвит иконку канала через общую
  карту `iconByThreadIcon` (как инбокс): `telegram` → самолётик, `whatsapp` →
  WhatsApp, неизвестная → облачко. task/email не тронуты.

## Грабли

- Триггер пикера по клавише — только в ProseMirror-keymap, не на wrapper-div.
- Любой плавающий попап композера, которому мешает `.composer-tools-cq`
  (container-query), — портить в `[data-composer-root]`.
- Иконка канала в поиске резолвится общей `iconByThreadIcon` (единый источник с
  инбоксом). Страница `/search` — свой локальный `EntityIcon`, здесь не затронута.
