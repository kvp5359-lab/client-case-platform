# Раскладка карточки проекта на досках: шаблон вплотную к названию

**Дата:** 2026-06-17
**Тип:** ui
**Статус:** completed (ждёт смок-теста в проде)

---

## Что было

В карточке проекта (строка на доске) поля шли в порядке
`иконка → название → дедлайн → шаблон`, причём шаблон прижимался к правому
краю. Название имело `flex-1` (grow) и растягивалось на всё свободное
место, визуально отрывая шаблон от названия.

## Что стало

Шаблон проекта переехал **влево, вплотную к названию**, дедлайн ушёл вправо.

- Порядок полей в дефолтной раскладке: `иконка → название → шаблон → дедлайн`
  (`cardLayoutUtils.ts`, `listSettingsConfigs.ts`).
- У названия убран `flex-1` — иначе оно растягивалось на всё свободное место
  и отрывало шаблон к правому краю (`BoardProjectRow.tsx`, `CardLayoutPreview.tsx`).
- Шаблону дан приоритет на сжатие `shrink-[3]` и `max-w-[40%]` — он жмётся
  первым, поэтому название всё равно получает приоритет по ширине.

## Файлы

- `src/components/boards/BoardProjectRow.tsx` — классы name/template.
- `src/components/boards/CardLayoutPreview.tsx` — превью раскладки (убран flex-1).
- `src/components/boards/cardLayoutUtils.ts` — порядок полей (template перед deadline).
- `src/components/boards/listSettingsConfigs.ts` — дефолтная раскладка карточки.
