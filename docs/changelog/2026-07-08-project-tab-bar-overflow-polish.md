# Ряд вкладок проекта: без пустоты + бутерброд на плашке

**Дата:** 2026-07-08
**Тип:** style (фронт)
**Статус:** ждёт деплоя фронта

---

Полировка ряда вкладок модулей проекта ([ProjectModuleTabBar.tsx](../../src/page-components/ProjectPage/components/ProjectModuleTabBar.tsx)) поверх измеряемого overflow (см. `2026-07-08-thread-template-project-overrides-and-project-tab-bar.md`).

- **Убрана пустота справа:** у `TabsList` снят `flex-1` → ряд по ширине контента, а не на всю строку. Кнопка-«бутерброд» встаёт **сразу за последней вкладкой** (когда всё влезло — без пустого промежутка до неё).
- **Бутерброд на той же серой плашке:** кнопка перенесена ВНУТРЬ `TabsList` (на му­тед-пилюлю), а не отдельным элементом на фоне страницы.
- **Тень-плашка:** кнопке добавлены `bg-background shadow` (белая приподнятая плашка, как активная вкладка), высота `h-7`, зазор `ml-0.5`, ховер `hover:text-foreground`.

Измеряемый overflow не затронут: замер идёт по ширине всего ряда (`rowRef`), «частичная» вкладка по-прежнему занимает остаток с обрезкой названия.
