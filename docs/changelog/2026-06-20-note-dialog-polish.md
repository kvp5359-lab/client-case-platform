# Окно заметки — крестик в углу + всегда видимая «Сохранить»

**Дата:** 2026-06-20
**Тип:** style (UX-полировка)
**Статус:** completed (ждёт деплоя CI)

---

Мелкая полировка окна текстовой заметки «Материалы команды»
(`ContextTextDialog`).

## Что сделано

- **Крестик закрытия** — кружок в верхнем правом **углу** окна
  (`-top-3 -right-3`, обводка + тень + hover-увеличение), как у попапа
  @-упоминаний. Встроенный крестик `DialogContent` скрыт. У окна снята
  обрезка (`!overflow-visible`), чтобы кружок выносился за угол; редактор
  внутри скроллится сам (своя `max-h`).
- **Кнопка «Сохранить»** — отображается **всегда**, но **неактивна**, когда
  сохранять нечего (`disabled={!isDirty || isSaving}`). Раньше появлялась
  только при наличии изменений.

## UI-примитив

- `DialogContent` получил необязательный проп `hideClose` — скрыть встроенный
  крестик, когда диалог рисует свой. Аддитивно, остальные диалоги не затронуты.
